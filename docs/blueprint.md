# **App Name**: P'tit Parlé (Little Speak)

## Core Features:

- Audio Word & Phrase Presentation: Presents core French vocabulary and simple phrases with clear, high-quality audio playback from native speakers, paired with relevant visual aids.
- Interactive Speech Practice: Enables children to record their voice repeating the presented French words and phrases directly through the browser's microphone for immediate practice and mimicking of native pronunciation.
- AI Pronunciation Assessment Tool: Utilizes the browser's native Web Speech API tool for real-time speech recognition, comparing a child's recorded pronunciation to the target native audio and providing basic visual feedback (e.g., a 'good job' indicator). This client-side approach is free, leveraging existing browser capabilities. However, its accuracy and language support can vary by browser and device compared to more robust, paid cloud services (like Google Cloud Speech-to-Text or Azure Cognitive Services), which offer higher accuracy and broader linguistic models but incur costs, typically with free tiers for initial usage.
- Listening Comprehension Games: Engaging, simple games where children listen to a French word or phrase and select the correct corresponding image or text from a set of options.
- Gamified Progress Dashboard: A visually appealing and child-friendly dashboard that tracks the child's learning journey, showcasing completed lessons, successful pronunciations, and earned 'stars' or 'badges' for motivation.

## Style Guidelines:

- Default Theme - Primary color: A warm and inviting medium blue (#5282E0) to evoke feelings of calm, creativity, and trust in a playful context. This provides a gentle contrast for elements on a lighter background.
- Default Theme - Background color: A very light, almost off-white with a subtle blue tint (#F0F1F5) to create a clean, airy, and friendly canvas for content, ensuring readability and comfort for young learners.
- Default Theme - Accent color: A bright, cheerful turquoise (#1AC5E6) used sparingly for interactive elements, highlights, and calls to action, drawing attention with its playful vibrancy while remaining harmonious with the primary blue.
- Pink Theme - Primary color: A soft, inviting medium pink (#F4B8CC) for a warm, friendly, and engaging feel, providing a gentle contrast against lighter backgrounds.
- Pink Theme - Background color: A very light, almost off-white with a subtle pink tint (#FDF6F8) to create a clean, airy, and friendly canvas for content, ensuring readability and comfort for young learners.
- Pink Theme - Accent color: A bright, cheerful fuchsia (#E6398F) used sparingly for interactive elements, highlights, and calls to action, drawing attention with its playful vibrancy while remaining harmonious with the primary pink.
- Body and headline font: 'PT Sans' (sans-serif) chosen for its humanist characteristics, providing a warm, clear, and modern readability that is suitable for both headings and body text, making it accessible for children.
- Utilize simple, friendly, and visually clear icons that are easily recognizable by young children. Icons should be illustrative of their actions or concepts (e.g., a microphone for recording, a speaker for playback).
- A clean and uncluttered layout with large, touch-friendly buttons and generous spacing to simplify navigation and reduce visual complexity for children. Information hierarchy will be clear and intuitive.
- Incorporate subtle and encouraging animations, such as gentle feedback on correct answers, visual cues for listening and speaking activities, and cheerful transitions to celebrate progress.
